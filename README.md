# calculator

Languages & Tools: HTML, CSS, Javascript.

When a user selects a number this is displayed in the output window as the "first number", when the user selects the operator (+, -, x, รท) this is displayed after the first number. The next number selected is displayed as the "second number", Both numbers are then calculated and the answer displayed when the equals button is selected. All numbers are converted to numbers rather than strings in order for the calculation to perform correctly.
The display window can then be cleared with the AC button.

used:
-DOM
-array iterator
-event listener
-if/else
